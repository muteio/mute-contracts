## Install

```bash
# Install project dependencies
npm install

# install OpenZeppelin SDK globally
npm install -g @openzeppelin/cli

# initialize repo with OpenZeppelin
oz init

# compile code
oz compile
```

## Testing

``` bash
# Run all unit tests
npm test

```
